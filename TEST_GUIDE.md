## ✅ 테스트 체크리스트

### 1. 스플래시 화면 (자동)
- [ ] 스플래시 화면 표시
- [ ] Ollama 엔진 상태: "준비 완료"
- [ ] Stable Diffusion: "미설치 (선택)"
- [ ] 자동으로 메인 화면으로 전환 (1-2초 후)

### 2. 프로젝트 생성
- [ ] "새 프로젝트 만들기" 버튼 클릭
- [ ] 모달 창 열림
- [ ] 제목 입력: "테스트 소설"
- [ ] 장르 선택: "소설 — 문학"
- [ ] "만들기" 클릭
- [ ] 토스트 메시지: "테스트 소설 프로젝트를 열었습니다"
- [ ] 자동으로 교열 탭으로 이동

### 3. 교열 기능
- [ ] 원고 입력창에 텍스트 입력:
  ```
  그는 공원에 앉아서 하늘을보았다. 구름이 천천히 움직이고있었다. 마음이 복잡했다. 결정을해야했다.
  ```
- [ ] 통계 표시 확인 (글자 수, 단어 수, 읽기 시간)
- [ ] "교열 시작" 클릭
- [ ] 로딩 표시 (1.5초)
- [ ] 결과 표시

**결과 탭 확인:**
- [ ] "교열된 원고" 탭: 수정된 텍스트 표시
- [ ] "수정 제안 (3)" 탭: 3개의 수정 제안
  - 띄어쓰기 오류
  - 문법 오류
  - 문체 개선
- [ ] "설정집" 탭: 인물, 장소 정보 표시

### 4. 교정 기능
- [ ] 사이드바 "교정" 클릭
- [ ] 원고 입력
- [ ] "교정 시작" 클릭

### 5. 원고 분석

- [ ] 결과 확인:
  - 장르 판별
  - 출판 가능성 (75%)
  - 개선 제안
  - 유사 도서

### 6. 마케팅 문구
- [ ] 사이드바 "마케팅 문구" 클릭
- [ ] 책 제목, 장르, 내용 입력
- [ ] "문구 생성" 클릭
- [ ] 결과 확인:

### 7. 표지 컨셉

### 8. 홍보 영상

## 🐛 디버깅

### 콘솔 에러 확인
1. 브라우저 개발자 도구 열기 (F12 또는 Cmd+Option+I)
2. Console 탭 확인
3. 에러 메시지가 있다면 복사

### 네트워크 에러 확인
1. Network 탭 확인
2. 빨간색(404, 500) 요청 확인
3. 파일 경로 문제 확인

### 일반적인 문제

**문제: 클릭이 안 됨**
- 해결: 콘솔에서 JavaScript 에러 확인
- `app.js` 로드 실패 → 경로 확인

**문제: 스타일이 안 됨**
- 해결: `css/style.css` 로드 확인
- Network 탭에서 404 확인

**문제: 모달이 안 열림**
- 해결: `openNewProject()` 함수 정의 확인
- 콘솔에 `openNewProject is not defined` 에러 확인

**문제: 아이콘이 안 보임**
- 해결: Tabler Icons CDN 로드 확인
- `css/style.css` 첫 줄 확인
